> **Partner with the Soul of Ceylon.** A premium digital experience for Sri Lanka's finest organic spices.

Daham Ceylon Spice is a high-performance, aesthetically pleasing web application designed for a premium spice exporter. It features a modern, responsive design that captures the essence of Sri Lankan heritage while providing a seamless B2B inquiry experience.

---

## ✨ Key Features

![Spice Market](https://images.unsplash.com/photo-1532336414038-cf19250c5757?auto=format&fit=crop&q=80&w=1200)

- 🚀 **Modern Tech Stack**: Built with React 19, Vite, and TypeScript for maximum speed and reliability.
- 🎨 **Premium Design**: A "Warm Organic" aesthetic using a custom color palette (Cream, Gold, Mustard, Dark) and serif typography.
- 🎭 **Immersive Animations**: Smooth scroll reveals and interactive elements powered by `motion/react`.
- 📧 **Direct Inquiries**: Integrated with FormSubmit.co for direct-to-inbox lead generation without a complex backend.
- 📱 **Fully Responsive**: Optimized for every device, from mobile phones to ultra-wide monitors.
- 🛠️ **Vercel Ready**: Pre-configured for seamless deployment as a serverless application.

---

## 🛠️ Tech Stack

- **Frontend**: [React 19](https://react.dev/), [React Router 7](https://reactrouter.com/)
- **Styling**: [Tailwind CSS 4](https://tailwindcss.com/)
- **Animations**: [Motion](https://motion.dev/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Backend (Dev)**: [Express](https://expressjs.com/)
- **Deployment**: [Vercel](https://vercel.com/) (Serverless Functions)

## 🚀 Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v18 or higher)
- [npm](https://www.npmjs.com/)

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/your-username/daham-ceylon-spice.git
   cd daham-ceylon-spice
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start the development server**
   ```bash
   npm run dev
   ```
   The app will be available at `http://localhost:3000`.

---

## 📦 Deployment

### Vercel (Recommended)

This project is pre-configured for Vercel. 

1. Push your code to GitHub.
2. Import the project into Vercel.
3. Vercel will automatically detect the settings and deploy.

**Note**: The first time you submit the contact form, you will receive an activation email from FormSubmit.co to verify your inbox.
